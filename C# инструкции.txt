mkdir <name> - создание каталога;

cd <name> - переход от каталога к каталогу;

dotnet new console - создание проекта на языке C#.(создаст файл program.cs 
и файл с настройкой)
для создания проекта в нужном каталоге нужно в нем находиться.
(обязательно нужно впихнуть в папку с проектом файл .gitignore 
снеобходимыми запретами, если работаешь с git)

Console.WriteLine(); - вывод данных в консоль;

dotnet run - команда для запуска кода; обязательно выполяется в каталоге, где был создан проект.

cd .. - переход в каталог повыше;

Типы данных в C#:
целочисленные типы данных:
int - целочисленный тип данных;
long - тоже целочисленный тип данных, который хранит большее число, чем int;

Вещественные типы данных:(позволяет хранить дробные числа)
float 
double - такое же отличие, как и  предыдущем пункте

Символьный тип данных:
char - для символов(буквы,цифры и т.д.)

Логический тип данных:
bool - true false
